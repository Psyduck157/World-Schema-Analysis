# SQL Master Study Guide 
This repository contains a deep dive into MySQL using the **World** database. 

### What's Inside:
- **Basics:** Data extraction, sorting, and filtering.
- **Advanced Logic:** Regex pattern matching and CASE statements.
- **Relational Power:** Inner Joins and Subqueries.
- **Analytics:** Aggregations with `GROUP BY` and `HAVING`.

### How to use:
Run the `.sql` file in MySQL Workbench or any SQL editor after loading the `world.sql` sample database.
